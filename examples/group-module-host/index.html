Group-aware Module Host

This simulates the Formant product container. It replies to request_devices and request_module_data with multi-device (group) payloads.
Tip: run examples/module-tester and paste its URL here.
In real coherence views, this would be the coherence group id (or whatever your host uses to define membership).
If the module calls App.refreshAuthToken(), this host will reply with auth_token containing this value.
