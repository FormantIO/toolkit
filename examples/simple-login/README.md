# Login Form

We provide a simple component ui-sdk-login to handle authorization for you if you just want to focus on user experience.

## Run locally

go to the project directory

```bash
  cd toolkit/examples/vr
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
