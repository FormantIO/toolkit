# URDF

This exaple shows how to use tranformation tree stream to create visual representation of the robot.

## Run locally

go to the project directory

```bash
  cd toolkit/examples/simple-urdf
```

Install dependencies

```
   npm i
```

Start the server

```bash
  npm run dev

```
